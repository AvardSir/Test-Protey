# Лекторий ПРОТЕЙ — регистрация на деловые мероприятия

Интерактивная форма регистрации на трёхдневный лекторий с выбором лекций по дням. Проект разработан для **НТЦ ПРОТЕЙ** — компании-разработчика телекоммуникационных решений и систем безопасности для критической информационной инфраструктуры (КИИ).

🔗 **Демо:** [ссылка на сайт](https://test-protey.vercel.app/)
---

## О проекте

Этот проект решает задачу удобного онлайн-участия в серии деловых лекций. Пользователь может:

- Просмотреть программу лектория с разбивкой по дням
- Выбрать интересующие лекции (один или несколько докладов)
- Заполнить форму регистрации с валидацией всех полей
- Получить обратную связь об успешной отправке заявки
- Симуляция и обработка неудачных запросов

## Технологии

| Категория | Технологии |
|-----------|-------------|
| **Фреймворк** | React 18 (функциональные компоненты + хуки) |
| **Сборщик** | Vite — быстрая разработка и сборка |
| **Формы и валидация** | React Hook Form — производительные формы с минимальным ререндером |
| **Стилизация** | Tailwind CSS — utility-first подход, кастомные градиенты |
| **TypeScript** | Полная типизация данных (лекции, формы, модальные окна) |
| **Деплой** | Vercel — автоматический CI/CD из GitHub |
| **Контроль версий** | Git |

---

## Ключевые возможности

- ✅ **Группировка лекций по дням** с аккордеоном (открытие/закрытие)
- ✅ **Множественный выбор лекций** с визуальной индикацией выбранных
- ✅ **Автоматическое форматирование телефона** + маска ввода
- ✅ **Валидация в реальном времени** с сообщениями об ошибках
- ✅ **Модальные окна** (успех / ошибка / не выбрана ни одна лекция)
- ✅ **Адаптивная вёрстка** под мобильные, планшеты и десктоп
- ✅ **Сохранение выбранных лекций** при повторном открытии (при необходимости можно добавить localStorage)

## Локальный запуск

```bash

npm install
npm run dev
```
